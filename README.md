A fun and interactive command-line game where the player tries to guess a secret number generated by the computer.

## Features
- **3 Difficulty Levels:** Easy (1-100), Medium (1-1000), and Hard (1-10000).
- **Attempt Tracking:** Counts how many tries it took to find the number.
- **Input Validation:** Ensures the user enters a valid number and doesn't crash the game.
- **Smart Feedback:** Tells the player if their guess is too high or too low.

## How to Play
1. Run the script: `python guessing_game.py`
2. Choose your difficulty level.
3. Keep guessing until you find the secret number!

## Lessons Learned
- Importing and using the `random` library.
- Working with `while` loops and `break` conditions.
- Type casting (`int()` and `str()`).
- Basic error handling with `isdigit()`.
